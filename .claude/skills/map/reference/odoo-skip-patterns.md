Skip these Odoo fields unless specifically needed:
- Chatter/activity: message_ids, activity_ids, etc.
- KPI/computed analytics: kpi_1month_mrr_delta, etc.
- Portal/website runtime: access_url, website_order_line, etc.
- Count fields: invoice_count, delivery_count, etc.
- Display fields: display_name, billing_period_display, etc.

Focus on: stored, writable, business-meaningful fields.
